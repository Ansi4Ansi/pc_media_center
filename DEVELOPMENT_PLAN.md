## Рекомендации для AI-агентов

1. **Начинать с Фазы 0** — она создаёт фундамент для всего
2. **Фазы 8, 9, 10 можно делать параллельно** с основной веткой (2→3→4→5→6→7)
3. **Каждая задача** — атомарная, можно отдать одному агенту
4. **После каждой фазы** — проверить компиляцию `flutter build <platform>`
5. **Кодогенерация** — запускать `dart run build_runner build` после изменений в таблицах/моделях
6. **Тесты** — писать параллельно с кодом, не откладывать на Фазу 11
7. **Читать ARCHITECTURE.md** перед началом каждой фазы — там структура и принципы
